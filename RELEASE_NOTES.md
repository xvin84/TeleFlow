# Release Notes

## [0.1.0-alpha] — 2026-03-17

Первый публичный релиз.

### Добавлено

**Авторизация и аккаунты**
- Авторизация через MTProto (Telethon): телефон → SMS-код → 2FA
- До 5 аккаунтов одновременно, быстрое переключение, статус online/offline
- Опциональный пароль приложения (bcrypt) с блокировкой UI при запуске
- Шифрование сессий: Fernet + PBKDF2 + уникальная соль на установку

**Шаблоны сообщений**
- CRUD шаблонов с форматированием (жирный, курсив, подчёркивание, зачёркнутый, моноширинный)
- Надёжное хранение code-форматирования через `QTextFormat.UserProperty`
- Медиавложения: до 10 файлов (изображения → album, прочее — поштучно)
- Импорт шаблонов из CSV

**Окно «Управление отправкой» (SendRulesDialog)**
- Единое окно для управления правилами отправки каждого шаблона
- Левая панель: назначение чатов с живым поиском и добавлением по клику
- Правая панель: расписания с поддержкой паузы, редактирования и удаления
- Кнопка «Отправить сейчас» внизу окна

**Планировщик**
- 5 режимов: один раз, ежедневно, по дням недели, с интервалом, случайное окно
- Персистентность задач (SQLite через APScheduler 4.x)
- Метаданные расписаний в `teleflow.db` (таблица `schedules`)
- FloodWait handler

**Интерфейс**
- Dark / Light тема, переключение без перезапуска
- Логи отправки (успех / ошибка / rate limit)
- Локализация: ru (основной), en (заготовка)
- CI: ruff + mypy на каждый push / PR

### Известные ограничения
- Системный трей и системные уведомления не реализованы
- Автозапуск не реализован
- CSV-визарды упрощены (нет маппинга колонок)
- Retry при `ConnectionError` отсутствует (только FloodWait)
- Windows: поддержка заявлена, тестировался минимально
